O mne

Volám sa Rudolf Kendra, študoval som na Žilinskej univerzite v Žiline.

Od apríla 2022 sa vo svojom voľnom čase venujem samo učeniu programovacieho jazyka JAVA. Následne od augusta 2022 do januára 2023 som absolvoval samo kurzy na stránke skillmea.sk, z ktorých som získal veľa hodnotných znalostí a zručností. Kurzy ako Java začiatočník, Java pokročilý, OOP, jazyk SQL a Spring boot. Všetko som si to chcel ešte poriadne ujasniť a preto som sa rozhodol prihlásiť na kurz Java developer, ktorý som začal v marci 2023 a úspešne ukončil v júny 2023, kde som si všetko skompletizoval vďaka môjmu lektorovi.
Na kurze sme znova prešli základy jazyku Java 17 a následne pokročilé veci Javy. Jazyk SQL, konkrétne MySQL a samozrejme našou náplňou boli projekty, na ktorých sme sa veľa naučili, k tomu sme okrem Javy využívali Maven, Spring boot, Hibernate atď.
